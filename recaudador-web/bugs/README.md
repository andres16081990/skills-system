# Bug Tracker — Agent System

## Cómo usar
1. Crea un archivo BUG-[ID].md en esta carpeta
2. Llena el template con el contexto del bug
3. Ejecuta /bugfix BUG-[ID] en Claude Code

## Template
```
# BUG-[ID]

## Descripción
Qué está pasando.

## Pasos para reproducir
1.
2.

## Comportamiento esperado
Qué debería pasar.

## Componentes a revisar
-

## Contexto adicional
PRs recientes, cuándo empezó, si es reproducible en local, etc.

## Prioridad
Alta / Media / Baja
```
